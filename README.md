# CodeAlpha Object Detection and Tracking

This project performs real-time object detection and tracking using YOLOv8 and OpenCV.

## Features

* Real-time webcam detection
* Object tracking
* Bounding boxes and labels
* YOLOv8 integration

## Technologies Used

* Python
* OpenCV
* YOLOv8
* Ultralytics
